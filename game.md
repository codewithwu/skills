# 📋 躲猫猫小游戏 - 完整需求规格

### 游戏概述
一款寻找隐藏白猫的视觉记忆与观察力解压小游戏。白猫在网格背景上奔跑后躲藏，玩家需要在限定时间内找出它们的藏身之处。

### 核心玩法流程

```
游戏开始 → 准备阶段(3秒) → 奔跑阶段(10秒) → 寻找阶段(30秒) → 结算
                ↓                    ↓
           黑白猫乱跑           白猫躲进方格
           黑猫继续跑           黑猫继续迷惑
```

### 详细设计

#### 1. 游戏配置
| 项目 | 设定 |
|-----|------|
| 网格大小 | 8×8 白色方格 |
| 白猫数量 | 3只 |
| 黑猫数量 | 3只 |
| 奔跑时间 | 10秒 |
| 寻找时间 | 30秒 |
| 方格样式 | 白底 + 浅灰边框，选中时高亮 |

#### 2. 游戏阶段

**阶段一：准备阶段（3秒倒计时）**
- 显示 "3, 2, 1, 开始！"
- 猫咪静止在随机位置

**阶段二：奔跑阶段（10秒）**
- 黑白猫在网格间随机移动（每0.3-0.5秒跳一格）
- 移动时有弹性动画
- 显示倒计时和提示文字："记住白猫的位置！"
- 黑猫和白猫都可见

**阶段三：寻找阶段（30秒）**
- 白猫消失（躲进当前所在方格）
- 黑猫继续在网格上奔跑（持续迷惑玩家）
- 玩家点击方格猜测白猫藏身位置
- 显示倒计时和剩余白猫数量
- 找到的白猫会重新出现并标记为"已找到"

**阶段四：结算**
- 显示结果：找到 X/3 只白猫
- 评价语（根据成绩变化）
- "再玩一次"按钮

#### 3. 交互细节

**点击反馈**
- 点击正确：白猫现身 + 金色粒子特效 + "喵~找到啦！" + 音效
- 点击错误：方格红色闪烁 + "这里没有猫" + 扣除5秒时间
- 已找到的方格不可重复点击

**视觉风格**
- 整体风格：清新简约、毛玻璃UI
- 主色调：白色+浅灰+淡金色点缀
- 猫咪设计：简约几何风格，黑猫有黄眼睛，白猫有蓝眼睛
- 动画：猫咪跳跃移动、方格呼吸态高亮

**辅助功能**
- 提示按钮：消耗5秒，高亮一个未找到白猫的候选区域（半透明光圈）
- 放弃按钮：直接显示所有白猫位置并结束游戏

#### 4. 难度平衡
| 难度因素 | 设定 |
|---------|------|
| 网格数量 | 8×8=64格 |
| 白猫比例 | 3/64 ≈ 4.7% |
| 干扰元素 | 3只持续移动的黑猫 |
| 记忆压力 | 需记住10秒内白猫的最终位置 |
| 容错惩罚 | 点错扣5秒 |

#### 5. 音效设计
- 背景音乐：轻快的8-bit循环（可选静音）
- 猫咪移动：轻柔的"哒"声
- 找到白猫："喵~" + 成功音效
- 点击错误：短促的"噗"声
- 倒计时最后5秒：心跳加速音效

### 扩展功能（可选）
- 难度选择：简单(6×6)/普通(8×8)/困难(10×10)
- 排行榜：记录最快找齐时间
- 成就系统：完美记忆（不错一次）、火眼金睛（30秒内完成）等
